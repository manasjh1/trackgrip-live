# TrackGrip Vision

Real-time Track Grip Analysis System powered by AI and Computer Vision.

## Project info

**TrackGrip Vision** is a high-tech dashboard for monitoring racetrack conditions, camera feeds, and telemetry data in real-time.

## Getting Started

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Install dependencies
npm install
# or
bun install

# Step 2: Start the development server
npm run dev
# or
bun run dev
